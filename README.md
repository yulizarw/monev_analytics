# Monitoring & Evaluation Analytics System

A Python-based analytics project for monitoring institutional performance, data completeness, and compliance in Monitoring & Evaluation (Monev) activities.

## Overview

This project processes institutional monitoring data to evaluate program performance, identify missing reports, measure compliance, and generate early warning indicators that support decision making.

The analysis focuses on improving reporting quality and identifying study programs requiring further intervention.

---

## Features

- Data cleaning and preprocessing
- Missing value analysis
- Compliance monitoring
- Data completeness evaluation
- KPI calculation
- Early Warning System (EWS)
- Study program performance ranking
- Composite Quality Index
- Statistical summaries
- Data visualization using Matplotlib

---

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib

---

## Workflow

1. Import monitoring data
2. Clean and standardize dataset
3. Analyze missing values
4. Measure reporting compliance
5. Calculate institutional KPIs
6. Build Early Warning indicators
7. Generate quality index for each study program
8. Visualize results
